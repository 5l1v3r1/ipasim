# Debugging inside a Docker container

```ps1
& 'C:\Program Files\Microsoft Visual Studio 15.0\Common7\IDE\Remote Debugger\x64\msvsmon.exe' /nostatus /silent /noauth /anyuser /nosecuritywarn
```
